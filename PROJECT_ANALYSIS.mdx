# LazyFit - Projekto Analizė

## Apžvalga

**LazyFit** yra visapusė fitnes ir sveikatingumo platforma, sukurta naudojant moderniausias web technologijas. Platforma apima mitybos planavimą, treniruočių programas, edukacinio turinio kursus ir narystės sistemą.

## Pagrindinės Funkcijos

### 1. Autentifikacija ir Vartotojų Valdymas
- Kredencialais paremta autentifikacija
- El. pašto patvirtinimas
- Slaptažodžio atstatymas
- Sesijų valdymas
- Avataro įkėlimas
- Paskyros ištrynimas

### 2. Mitybos Sistema
- Maisto produktų duomenų bazė
- Kalorijų skaičiuoklė (BMR, TDEE)
- Mitybos planų kūrimas
- Asmeniniai mitybos planai
- Valgių sekimas
- Makroelementų stebėjimas

### 3. Receptų Sistema
- Receptų kūrimas ir valdymas
- Kategorijos
- Ingredientų valdymas
- Mėgstami receptai
- Sunkumo lygiai
- Paruošimo/gaminimo laikas

### 4. Treniruočių Sistema
- Pratimų biblioteka
- Treniruočių kūrimas
- Programų organizavimas
- Periodizacija
- Supersetų valdymas
- Raumenų grupių sekimas

### 5. Kursų Sistema
- Edukacinių kursų kūrimas
- Video ir tekstinės pamokos
- Sunkumo lygiai
- Pamokų pertvarkymas

### 6. Narystės Sistema
- Narystės planų valdymas
- Kainodara su nuolaidomis
- Prieigos kontrolė pagal funkcijas
- Sąskaitų generavimas

### 7. Turinio Valdymas
- Dinaminių puslapių kūrimas
- Meniu valdymas
- El. pašto šablonai
